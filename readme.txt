Duplicate any post, page, or custom post type in one click — including taxonomies, meta, and featured images.

== Description ==
QuickClone WP adds a simple “Duplicate” link to your WordPress admin post list, letting you instantly clone posts, pages, or any custom post type.

== Features ==
* One-click duplication for posts, pages, and CPTs
* Copies all taxonomies and custom fields
* Retains featured image
* Lightweight and fast
* Secure, using nonces and capability checks

== Installation ==
1. Upload the `quickclone-wp` folder to `/wp-content/plugins/`
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Hover over a post or page in the list and click “Duplicate”

== Frequently Asked Questions ==
= Does it copy custom fields? =
Yes. All post meta are copied, except for internal WordPress meta (like `_edit_lock`).

= Does it support custom post types? =
Yes. Any public post type can be duplicated.
